# Canvas

# 简介

最初由 Apple 使用在 Safari 中，后来 Firefox/Chrome 分别实现，经过一段时间发展，成为 H5 标准。 Canvas 可理解为画布，栅格化的。

# 使用场景

小游戏、头像裁剪、Charts
